かわいいデザインで
ブランチを練習しよう

サークルや友達とのコラボにも使える、シンプルで可愛いテンプレート。

ステップ1

新しいブランチを作成して、好きな変更をしてみましょう。

ステップ2

プルリクを作成して、レビューをもらおう。

ステップ3

マージしてシェア！みんなで協力して楽しく練習。

デモ

ボタンを押すと、メッセージとアニメーションが表示されます。

推奨コマンド

git checkout -b feature/my-change
# 編集 -> git add . -> git commit -m "Add cute UI"
git push -u origin feature/my-change
