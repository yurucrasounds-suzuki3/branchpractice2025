Branch Practice

このリポジトリでブランチ運用を練習しましょう。

使い方

  1. ローカルでサーバーを起動してページを表示します。
  2. 新しいブランチを作ってファイルを編集し、プルリクを作成します。

デモ

以下のボタンを押すと、ブランチ練習用のメッセージが表示されます。

推奨コマンド

git checkout -b feature/my-change
# 編集 -> git add . -> git commit -m "Add demo"
git push -u origin feature/my-change
